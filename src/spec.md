# Giant Banana Landing Page

## Overview
A minimalist landing page that displays a single, centered, full-width image of a giant banana with no additional UI elements.

## Features
- Display a large, centered banana image that spans the full width of the viewport
- Clean, minimalist background design
- No text, buttons, navigation, or other UI elements
- Responsive design that works across different screen sizes

## Technical Requirements
- No backend functionality required - this is a static frontend-only application
- No data persistence needed
- No user interactions or dynamic content
